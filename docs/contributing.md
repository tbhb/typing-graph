# Contributing

Thank you for your interest in contributing to typing-graph.

## Current status

!!! note "Not yet accepting pull requests"

    typing-graph is in early development and is not yet accepting pull requests.

The project is currently establishing its core architecture, API design, and testing infrastructure. This approach ensures a solid foundation before opening up to external contributions.

## How you can help now

While the project does not accept code contributions yet, you can still help:

<div class="grid cards" markdown>

-   :material-bug:{ .lg .middle } **Report bugs**

    ---

    If you encounter issues while using typing-graph, please open a [GitHub issue](https://github.com/tbhb/typing-graph/issues) with a minimal reproducible example.

-   :material-lightbulb:{ .lg .middle } **Request features**

    ---

    Have an idea that would make typing-graph more useful? Open an issue to start a discussion.

-   :material-file-document-edit:{ .lg .middle } **Improve documentation**

    ---

    Found a typo or unclear explanation? Documentation improvements are welcome.

-   :material-message-text:{ .lg .middle } **Share feedback**

    ---

    Open an issue or [discussion](https://github.com/tbhb/typing-graph/discussions) about how you're using typing-graph and what could be better.

</div>
