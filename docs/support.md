# Getting help

Whether you're just getting started or working on an advanced use case, these resources can help.

## Documentation

The [usage documentation](index.md) covers installation, core concepts, and common patterns. The [API reference](reference/api.md) provides detailed information on all public classes and functions.

## GitHub discussions

For questions, ideas, or general discussion, use [GitHub Discussions](https://github.com/tbhb/typing-graph/discussions). Your question and its answer help others in the community.

## GitHub Issues

Found a bug or have a feature request? Open an [issue on GitHub](https://github.com/tbhb/typing-graph/issues). Please search existing issues first to avoid duplicates.

## Stack overflow

You can also ask questions on [Stack Overflow](https://stackoverflow.com/questions/tagged/typing-graph) using the `typing-graph` tag. Note that this is not always monitored by the maintainers.
